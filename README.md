# Lumen — Modern Landing Page

A fully responsive, modern landing page built with **HTML5, CSS3, and vanilla JavaScript** only.
No frameworks, no build step, no dependencies.

## Structure

```
project/
├── index.html      # Semantic HTML5 markup
├── style.css       # All styling (custom properties, Flexbox, Grid, media queries)
├── script.js       # Mobile nav toggle, form feedback, footer year
├── images/         # Local image assets (placeholder URLs used by default)
└── README.md
```

## Run

Just open `index.html` directly in any modern browser — that's it. No server, no install.

## Features

- Semantic HTML5 (`header`, `nav`, `main`, `section`, `article`, `footer`)
- Clean CSS3 with custom properties, Flexbox & Grid
- Sticky navigation with mobile hamburger menu (vanilla JS)
- Hero with background image and dual CTAs
- About, Services (4 cards), Gallery (6 images), Features, Contact form
- Contact form with client-side validation feedback (vanilla JS)
- Soft shadows, rounded corners, smooth hover effects
- Blue + white palette, Inter + Plus Jakarta Sans typography
- Fully responsive: desktop, tablet, mobile
- Accessible: alt text, form labels, semantic structure, ARIA attributes

## Customization

- Edit colors and spacing in the `:root` block at the top of `style.css`.
- Drop your own images into `images/` and update the `src` attributes in `index.html`.
